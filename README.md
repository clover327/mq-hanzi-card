# 字源识字卡生成器（Claude Code Skill）

输入一个汉字，生成一张融合**左民安字源考据**与**魏老师识字方法论**的儿童字源识字卡（HTML）。

不是机械识字卡。用字源帮孩子理解一个字，用字族帮孩子迁移一批字，用语境帮孩子把字用活，用朗读帮孩子走向阅读自动化。

## 两个版本

| 版本 | 说明 | 适合 |
|------|------|------|
| **精简版（纯文字）** | 纯文字 + CSS 排版，不需要图片生成能力 | 大多数用户，开箱即用 |
| **完整版（含AI插画）** | 包含 Gemini Image API 生成头图插画 | 有 API Key 的用户 |

## 安装方式

### 精简版（推荐）

1. 将 `精简版（纯文字）/` 文件夹整个复制到你的 Claude Code skills 目录：
   ```
   ~/.claude/skills/mq-hanzi-card/
   ```
2. 确保目录结构为：
   ```
   ~/.claude/skills/mq-hanzi-card/
   ├── SKILL.md
   └── references/
       ├── hanzi_freq.csv
       ├── json-schema.md
       ├── spec-v1.1.md
       └── wei-methodology.md
   ```
3. 重启 Claude Code，对话中输入汉字即可生成字卡

### 完整版（含 AI 插画）

1. 将 `完整版（含AI插画）/` 文件夹整个复制到 skills 目录（同上）
2. 打开 `scripts/image_generator.py`，将 API Key 替换为你自己的：
   ```python
   API_KEY = "你的API_KEY（请替换为自己的Key）"
   ```
3. API 使用的是 Gemini Image Pro 模型

## 使用方式

对话中说以下任何一种即可触发：
- 「帮我做一张 山 的字卡」
- 「字卡：水」
- 「生成 北 的识字卡」
- 直接发一个汉字

## 字卡包含什么

每张卡包含 10 个模块：

1. **Hero 区域** — 大字、拼音、造字类型、笔画数、字频排名
2. **字源故事** — 用儿童语言讲字的由来（80-180字）
3. **字形演变** — 汉字叔叔外链，看真实古字形
4. **意义旅行** — 字义发展链式结构
5. **字族/偏旁/亲戚字** — 高频词族、偏旁家族、声旁家族、字源亲戚、易混字
6. **字→词→句** — 阅读阶梯，递进 + 复现
7. **我来读** — 听我读、跟着读、我来回答（录音模块）
8. **亲子互动** — 家长可立刻执行的小互动
9. **爸妈3分钟带法** — 不讲大道理，只讲步骤
10. **来源说明** — 字源考据与方法论出处

## 方法论来源

- **左民安《细说汉字》** — 字源依据
- **魏老师识字方法论** — 教学组织逻辑
